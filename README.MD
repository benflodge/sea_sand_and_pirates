# Sea, Sand and Pirates
A simple JavaScript canvas pirate ship shooter game

## Demo
https://benflodge.github.io/sea_sand_and_pirates/

## Controls
* Up - Sails Up
* Down - Sails Down
* Left - Turn to Port
* Right - Turn to Starboard
* C - Fire Cannon Port
* V - Fire Cannon Starboard

# Development
To run in an development environment
```npm install```
```npm run dev```
```npm start```
